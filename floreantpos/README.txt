1. Create FloreantPOS database: (in sample-db folder)
	unzip posdb.zip

2. Start FloreantPOS database: (in sample-db folder)
	start-server.bat

3. Update FloreantPOS database with Scipio info:
	update_posdb.bat

4. Start Floreant POS UI:
	floreantpos.bat


Note:  Steps 1 & 3 only need to be run one time to initialize the database.
